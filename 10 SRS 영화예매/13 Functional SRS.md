# 온라인 영화 예매 시스템 — 기능적 요구사항 요약

## 1. 사용자 관리 기능

* 회원 가입 및 로그인 지원
* OAuth 기반 소셜 로그인 지원
* JWT 기반 인증 처리
* 사용자 프로필 및 예매 내역 조회

---

## 2. 영화 조회 기능

* 현재 상영 영화 목록 조회
* 영화 상세 정보 조회
* 영화 검색 기능 제공

  * 제목
  * 장르
  * 배우 기준 검색

---

## 3. 극장 및 상영 일정 기능

* 지역별 극장 조회
* 영화별 상영 일정 조회
* 상영관 및 좌석 배치도 조회

---

## 4. 좌석 예약 기능

* 실시간 좌석 상태 조회
* 좌석 임시 선점(Hold)
* 중복 좌석 예약 방지
* 좌석 점유 시간 만료 처리
* 결제 완료 후 예약 확정

---

## 5. 결제 기능

* 카드 및 간편결제 지원
* 외부 PG 연동
* 결제 승인/실패 처리
* 결제 실패 시 좌석 자동 반환
* 예매 취소 및 환불 처리

---

## 6. 알림 기능

* 예매 완료 알림 발송
* 결제 실패 알림 발송
* SMS / 이메일 / Push 알림 지원

---

## 7. 관리자 기능

* 영화 등록 및 수정
* 상영 일정 관리
* 상영관 및 좌석 정보 관리
* 예약/결제 상태 모니터링

---

## 8. 이벤트 기반 처리 기능

* 서비스 간 이벤트 기반 비동기 통신
* 좌석 선점 이벤트 처리
* 결제 완료 이벤트 처리
* 예약 확정 이벤트 처리
* 알림 발송 이벤트 처리

---

## 9. 핵심 비즈니스 흐름

### 영화 예매 흐름

1. 영화 조회
2. 상영 일정 선택
3. 좌석 조회
4. 좌석 선점
5. 결제 진행
6. 예매 확정
7. 알림 발송

### 예매 취소 흐름

1. 예약 조회
2. 예매 취소 요청
3. 환불 처리
4. 좌석 반환
5. 취소 알림 발송

---

## 10. 핵심 도메인 특징

* 실시간 좌석 선점 처리
* 동시성 제어 필요
* 결제와 예약 상태 연동
* 이벤트 기반 상태 전이 관리
* Eventually Consistent 구조 지원
* 장애 발생 시 보상 처리(Saga) 필요
